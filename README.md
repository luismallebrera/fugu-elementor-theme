# Fugu Elementor Theme

A minimal, Elementor-ready WordPress starter theme for building beautiful websites with Elementor.

## What's Included

This repository contains:

- **fugu-elementor-theme/** - The complete WordPress theme
  - Full Elementor Theme Builder support
  - Pre-built demo Elementor kit with colors and typography
  - Homepage template for quick start
  - Responsive and lightweight design

## Quick Start

1. Download the latest release from [Releases](../../releases/latest)
2. Install the theme in WordPress (see [Installation Guide](fugu-elementor-theme/README.md))
3. Import the demo Elementor kit and templates
4. Start building your site!

## Documentation

Full documentation is available in the theme's [README.md](fugu-elementor-theme/README.md).

## Building the Theme ZIP

To create a distributable ZIP file:

```bash
cd fugu-elementor-theme/bin
./create-zip.sh
```

This will create `fugu-elementor-theme.zip` ready for distribution.

## Requirements

- WordPress 5.0+
- PHP 7.4+
- Elementor plugin (free or Pro)
